## Conexiones

### Oracle

- OPTIONS > JDBC URL: `jdbc:oracle:thin:oracle:1521/XE`
- OPTIONS > USER:PASSWORD : `system:oracle`
- DRIVER PROPERTIES > `oracle.jdbc.timezoneAsRegion: false`
